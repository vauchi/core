# MVP Success Criteria

## Functional
- [x] Create identity
- [x] Add/edit/remove fields
- [x] Generate QR code
- [ ] Scan QR code (camera)
- [x] Bidirectional exchange
- [x] Sync updates via relay
- [x] Visibility rules enforced
- [x] Backup/restore identity
- [x] Search contacts

## Performance
- [ ] App start < 2s
- [ ] Exchange < 5s
- [x] Offline queuing works
- [ ] Battery-efficient sync

## Security
- [x] Data encrypted at rest
- [x] E2E encryption
- [x] Forward secrecy
- [ ] Keys in Android Keystore

## Launch
- [ ] Release signing
- [ ] ProGuard enabled
- [ ] Relay deployed
- [ ] Play Store listing
- [ ] Privacy policy
