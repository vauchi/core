# Launch Success Criteria

## Functional (All Platforms)
- [x] Create identity
- [x] Add/edit/remove fields
- [x] Generate QR code
- [x] Scan QR code (camera)
- [x] Bidirectional exchange
- [x] Sync updates via relay
- [x] Visibility rules enforced
- [x] Backup/restore identity
- [x] Search contacts

## Platform Coverage
- [x] Android app
- [ ] iOS app
- [ ] Desktop app (Tauri)
- [ ] Multi-device sync / Device linking

## Performance
- [ ] App start < 2s
- [ ] Exchange < 5s
- [x] Offline queuing works
- [ ] Battery-efficient sync

## Security
- [x] Data encrypted at rest
- [x] E2E encryption
- [x] Forward secrecy
- [ ] Keys in platform secure storage (Keystore/Keychain)
- [ ] Security audit completed

## UI/UX
- [ ] UI/UX review completed
- [ ] Accessibility basics (contrast, touch targets)

## Launch
- [ ] Release signing (Android + iOS)
- [ ] ProGuard/R8 enabled
- [ ] Relay deployed (production)
- [ ] App store listings (Play Store + App Store)
- [ ] Privacy policy
- [ ] Desktop distribution (website/GitHub releases)
