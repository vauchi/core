# Vauchi Project Summary

Privacy-focused contact card exchange. Users exchange cards in-person via QR, control what each contact sees, and changes sync automatically.

## Key Features

- **Physical exchange only** - QR code scan required (no remote harvesting)
- **Field-level visibility** - Control what each contact sees
- **Real-time sync** - Updates propagate via encrypted relay
- **E2E encrypted** - Relay sees only encrypted blobs
- **Multi-device sync** - Link devices via QR, keep contacts in sync
